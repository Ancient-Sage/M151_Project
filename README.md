# M151_Project
Modul 151 Mastermind Projekt mit Vue.js

## Frontend Starten:
Im M151_Project Ordner:
##### cd frontend
##### npm run serve

---

## Backend Starten:
Im M151_Project Ordner:
##### cd backend
##### nodemon

---

### Zum Löschen der Datenbank-Einträge:
Im M151_Project Ordner:
##### cd backend
##### npm run repl
##### await knex('score').del()
